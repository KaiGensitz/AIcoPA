# Legacy synthetic/template files

This folder contains legacy synthetic/demo survey artifacts that were previously
stored in the repository root. They are kept here only as clearly labelled
synthetic examples/templates for development and testing.

Do not replace these files with real LimeSurvey exports or participant data.
Real raw exports, sensitive mapping/contact files, participant imports,
pseudonymized outputs, and QC reports must be stored outside the repository or
in ignored secure folders.
